# 🔍 Поиск и бронирование туров - Инструкция

## ✅ Что реализовано

Полный цикл работы с турами:
1. **Поиск туров** с фильтрами
2. **Результаты поиска** с пагинацией
3. **Детальная страница тура**
4. **Бронирование тура**
5. **Подтверждение бронирования**

---

## 🚀 Как пользоваться

### 1. Поиск туров

**URL**: http://malaktour.test/search/tours

#### Доступные фильтры:
- **Направление**: Страна, Курорт
- **Вылет из**: Город вылета
- **Даты**: Дата вылета (от/до), Дата возврата
- **Продолжительность**: Количество ночей (от/до)
- **Гости**: Взрослые, Дети
- **Цена**: Диапазон цен, Валюта
- **Отель**: Категория (звезды)
- **Питание**: Тип питания (BB, HB, FB, AI)
- **Тип тура**: Beach Holiday, Excursion, Combined
- **Программа**: Standard, VIP
- **Транспорт**: Самолет, Автобус
- **Дополнительно**:
  - ✨ Hot Deals (горящие туры)
  - ⚡ Instant Confirmation
  - ✓ No Stop Sale

#### Как искать:
1. Выберите страну (обязательно)
2. Укажите желаемые параметры
3. Нажмите **Search Tours**
4. Получите результаты с возможностью сортировки

---

### 2. Результаты поиска

**URL**: http://malaktour.test/search/tours (POST)

#### Что вы увидите:
- **Карточки туров** с основной информацией:
  - Фото отеля (placeholder)
  - Название отеля + звезды (⭐⭐⭐⭐⭐)
  - Страна и курорт
  - Даты заезда
  - Количество ночей
  - Тип питания
  - Цена с валютой
  - Бейджи: 🔥 Hot Deal, ⚡ Instant Confirmation

- **Действия**:
  - **View Details** - детальная информация
  - **Book Now** - сразу забронировать

- **Боковая панель** с фильтрами (можно изменить поиск)
- **Пагинация** (20 туров на страницу)
- **Счетчик**: показывает количество найденных туров

---

### 3. Детальная страница тура

**URL**: http://malaktour.test/tours/{id}

#### Информация:
- **Hero секция**:
  - Название отеля с категорией
  - Местоположение (страна, курорт)
  - Бейджи (Hot Deal, Instant Confirmation)

- **Быстрая информация**:
  - Tour Type (тип тура)
  - Program (программа)
  - Transport (транспорт)
  - Meal Plan (питание)

- **Вкладки**:
  - **Details**: даты, длительность, город вылета, количество гостей
  - **Hotel Info**: описание отеля, адрес, удобства
  - **Conditions**: условия бронирования, политика отмены

- **Booking Card** (sticky sidebar):
  - Цена за тур
  - Валюта
  - Кнопка **Book This Tour**

---

### 4. Бронирование тура

**URL**: http://malaktour.test/tours/{id}/book

#### Форма бронирования состоит из:

##### Tour Summary
- Краткая информация о туре
- Отель, направление, даты

##### Contact Information
- **Full Name** (обязательно)
- **Email** (обязательно)
- **Phone** (обязательно)

##### Traveler Information
Динамическая форма для каждого туриста:
- **First Name** (обязательно)
- **Last Name** (обязательно)
- **Birth Date** (обязательно)
- **Gender** (Male/Female)
- **Passport Number**
- **Passport Expiry Date**
- **Nationality**

**Кнопки**:
- ➕ Add Traveler - добавить туриста
- ➖ Remove - удалить туриста

##### Additional Information
- **Special Requests / Notes** - дополнительные пожелания

##### Terms & Conditions
- ☑ Checkbox для подтверждения согласия

##### Price Summary (sidebar)
- Автоматический расчет: цена × количество туристов
- Отображение валюты
- Итоговая сумма

#### Кнопки:
- **Complete Booking** - завершить бронирование
- **Cancel** - отменить

---

### 5. Подтверждение бронирования

**URL**: http://malaktour.test/bookings/{id}/confirmation

#### Что вы увидите:
- ✅ **Success message** - бронирование успешно создано

##### Booking Details
- **Order Number**: уникальный номер (ORD-YYYYMMDD-XXXXXX)
- **Status**: Pending (в ожидании подтверждения)
- **Booking Date**: дата создания
- **Total Amount**: итоговая сумма

##### Tour Information
- Название отеля с категорией
- Направление (страна, курорт)
- Даты заезда/выезда
- Длительность
- Тип питания

##### Travelers Information
Список всех туристов с:
- Имя, фамилия
- Дата рождения, пол
- Паспортные данные

##### Next Steps
Информация о том, что будет дальше:
1. Наш менеджер свяжется с вами в течение 24 часов
2. Подтверждение бронирования отправлено на email
3. Проверьте статус в разделе Claims

##### Action Buttons
- **View Order Details** - перейти к заявке
- **Search More Tours** - искать другие туры
- **Go to Dashboard** - на главную

## 📊 Пример использования

### Шаг 1: Поиск
```
1. Откройте http://malaktour.test/search/tours
2. Выберите страну: Turkey
3. Выберите курорт: Antalya
4. Укажите даты: с сегодня на 7 дней
5. Количество гостей: 2 взрослых
6. Нажмите Search Tours
```

### Шаг 2: Результаты
```
Вы увидите 10 туров в Анталию
- Цены от $1200 до $2100
- Отели 4-5 звезд
- All Inclusive
- Некоторые с бейджем "Hot Deal"
```

### Шаг 3: Просмотр деталей
```
Нажмите "View Details" на любом туре
Изучите:
- Детали тура
- Информацию об отеле
- Условия бронирования
```

### Шаг 4: Бронирование
```
1. Нажмите "Book This Tour"
2. Заполните контактную информацию
3. Добавьте информацию о туристах (2 человека)
4. Добавьте пожелания (опционально)
5. Согласитесь с условиями
6. Нажмите "Complete Booking"
```

### Шаг 5: Подтверждение
```
Получите номер заказа (например: ORD-20260204-123456)
Проверьте детали бронирования
Перейдите к заявкам для отслеживания статуса
```

## 💡 Советы

### Для поиска:
- Используйте фильтр "Hot Deals" для скидок
- "Instant Confirmation" - туры с мгновенным подтверждением
- Диапазон цен помогает найти оптимальные варианты
- Можно искать без указания точных дат (выберите диапазон)

### Для бронирования:
- Паспортные данные не обязательны, но желательны
- Можно добавить столько туристов, сколько нужно
- Special requests попадают в заметки к заказу
- После бронирования менеджер свяжется с вами

### Проверка статуса:
- Все бронирования доступны в разделе Claims
- Номер заказа можно использовать для отслеживания
- Status показывает текущее состояние заявки

## 🔧 Техническая информация

### Валидация:
- Email должен быть корректным
- Все обязательные поля отмечены *
- Дата рождения должна быть в прошлом
- Срок действия паспорта - в будущем

### Расчет цены:
```
Итоговая цена = Цена тура × Количество туристов
```

### Формат номера заказа:
```
ORD-YYYYMMDD-XXXXXX
где:
- YYYYMMDD - дата создания
- XXXXXX - случайное число
```

### Статусы бронирования:
- **Pending** - ожидает подтверждения
- **Confirmed** - подтверждено
- **Cancelled** - отменено

## 🐛 Если что-то не работает

### Проблема: Не отображаются результаты поиска
**Решение**: Убедитесь, что в базе есть туры. Запустите:
```bash
php artisan db:seed --class=BasicDataSeeder
```

### Проблема: Ошибка при бронировании
**Решение**: Проверьте, что вы авторизованы и ваше агентство активно

### Проблема: Не загружаются изображения отелей
**Решение**: Это нормально - используются placeholder изображения. Позже можно добавить реальные фото
