## 🛠️ Полезные команды

### Просмотр данных в базе
```bash
php artisan tinker
```

Примеры команд в tinker:
```php
// Все страны
\App\Models\Country::all();

// Все туры
\App\Models\Tour::all();

// Подсчет отелей
\App\Models\Hotel::count();

// Найти тур по ID
\App\Models\Tour::find(1);

// Все валюты
\App\Models\Currency::all();
```

### Добавить больше данных
```bash
# Запустить seeder снова
php artisan db:seed --class=BasicDataSeeder

# Или пересоздать базу и заполнить заново
php artisan migrate:fresh --seed
```

### Очистить кеш
```bash
php artisan optimize:clear
```

### Просмотр маршрутов
```bash
php artisan route:list
```

## 🔐 Тестовые пользователи

На данный момент тестовых пользователей нет. Создайте своего через регистрацию.

Если хотите создать тестового пользователя через tinker:
```php
$agency = \App\Models\Agency::create([
    'name' => 'Test Travel Agency',
    'phone' => '+998901234567',
    'email' => 'test@agency.com',
    'is_active' => true
]);

$user = \App\Models\User::create([
    'agency_id' => $agency->id,
    'name' => 'Test User',
    'email' => 'test@example.com',
    'password' => bcrypt('password'),
    'phone' => '+998901234567',
    'is_active' => true
]);

// Назначить роль администратора
$user->assignRole('administrator');
```

Логин:
- **Email**: test@example.com
- **Password**: password

## 🐛 Если что-то не работает

### 1. Проверьте логи
```bash
tail -f storage/logs/laravel.log
```

### 2. Очистите кеш
```bash
php artisan optimize:clear
```

### 3. Проверьте миграции
```bash
php artisan migrate:status
```

### 4. Проверьте данные
```bash
php artisan tinker
> \App\Models\Tour::count()
> \App\Models\Country::count()
```

### 5. Пересоздайте базу (если нужно)
```bash
php artisan migrate:fresh
php artisan db:seed --class=BasicDataSeeder
```
