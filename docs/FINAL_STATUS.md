## 🔄 Полный цикл работы

### Тестовый сценарий:

```
1. Регистрация:
   → http://malaktour.test/register
   → Создать агентство + пользователя

2. Вход:
   → http://malaktour.test/login
   → Email + Password

3. Dashboard:
   → Просмотр статистики
   → Быстрые действия

4. Поиск туров:
   → http://malaktour.test/search/tours
   → Выбрать фильтры (страна: Turkey)
   → Нажать Search Tours

5. Результаты:
   → Просмотр 10 туров в Анталию
   → Выбрать понравившийся тур
   → Нажать "View Details"

6. Детали тура:
   → Изучить информацию
   → Нажать "Book This Tour"

7. Бронирование:
   → Заполнить контактную информацию
   → Добавить туристов (2 человека)
   → Нажать "Complete Booking"

8. Подтверждение:
   → Получить номер заказа (ORD-20260204-123456)
   → Просмотреть детали
   → Перейти к заявкам

9. Мои заявки:
   → http://malaktour.test/claims
   → Найти свою заявку по номеру
   → Проверить статус (Pending)
```

## 💻 Команды для работы

### Запуск проекта
```bash
# Открыть в браузере
http://malaktour.test

# Или через artisan
php artisan serve
```

### База данных
```bash
# Пересоздать базу
php artisan migrate:fresh

# Заполнить тестовыми данными
php artisan db:seed --class=BasicDataSeeder

# Tinker для работы с данными
php artisan tinker
```

### Проверка
```bash
# Маршруты
php artisan route:list

# Статус миграций
php artisan migrate:status

# Информация о проекте
php artisan about
```

### Очистка
```bash
# Очистить весь кеш
php artisan optimize:clear

# Очистить только config
php artisan config:clear

# Очистить только view
php artisan view:clear
```
